# AI 101 – Practical Guide to AI Assistants

![AI 101 course logo](logo.png)

This repository contains a **practical introduction to working with AI assistants and Large Language Models (LLMs)** in modern development environments.

The guides demonstrate how to set up a hybrid AI workflow that combines **local AI models** with **cloud-based models**, and how to interact with them effectively using prompt engineering and AI agents.

The examples use the following technologies:

- Ollama (local AI models)
- Azure AI Foundry / Azure OpenAI
- Visual Studio Code
- Continue AI extension

## Guides

- [Part 1 – AI Initial Setup & Deployment](01-init/README.md)
- [Part 2 – Prompt Engineering Basics](02-base/README.md)
- [Part 3 – AI Agents and Development Workflows](03-continue/README.md)

## Goal

The goal of this repository is to provide a **hands‑on starting point for developers and technical teams** who want to integrate AI tools into their daily workflows.

By following these guides you will learn how to:

- set up an AI development environment
- run local and cloud AI models
- write effective prompts
- work with AI coding assistants

## Disclaimer

Large language models may generate incorrect or fabricated information.  
Always verify important outputs and validate results when using AI systems.
